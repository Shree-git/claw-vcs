# Demo Media

Small checked-in launch media:

- [basic-demo.cast](basic-demo.cast): short asciinema v2 recording of the core
  intent, change, snapshot, ship, integrate, and status path. The maintained
  script in `examples/basic-demo/scripts/demo.sh` is the source of truth for the
  complete current demo.
- [status-screenshot.svg](status-screenshot.svg): terminal-style `claw status`
  screenshot.
- [ship-capsule-screenshot.svg](ship-capsule-screenshot.svg): terminal-style
  `claw ship` and capsule output screenshot.
- [command-gallery.svg](command-gallery.svg): terminal-style screenshots for
  `claw status`, `claw log`, `claw diff`, `claw show`, `claw policy`, failed
  `claw integrate`, successful `claw ship`, and `claw git-export`.

Recommended external launch media:

- short GIF showing `claw status`, `claw log`, policy creation, capsule
  verification, and Git export
- longer narrated recording of `examples/basic-demo/scripts/demo.sh`
- social preview image generated from the landing page message

Store large raw recordings outside the repository. Large captures belong in
release assets or docs hosting.
